# JavaFinalProject_Nacua_Kinsley

"Student Grade Evaluation System"

Problem Statement:
This program is design to calculate the average grade and decide if the student has passed or failed.
The project will dispaly the means for the user to enter multiple subject grades, calculate the average, and display the appropriate markings through a Java application.

Program Features:
1. Scanner - it reads and accerpts what the user inputs.
2. Allows the user to interact with the program like input grades.
3. Stores the grade in the Array.
4. Calculate the average grade of the user.
5. Display marks whether (FAILED OR PASSED) based on the average grade.
6. Implemet OOP (Object Oriented Programming) concept.
7. It uses Loops, Methods and Decision Structure.

How to Run the Program:
1. Open IntelliJ IDEA (I used IntelliJ IDEA) you can used any java software program you want.
2. Create new java project
3. Make a new file name it Main.java
4. Copy and Paste my code into your file.
5. Then Click RUN.
